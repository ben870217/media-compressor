## ✨ 核心特性

- 🎥 **純前端硬體加速轉碼**：基於 `mediabunny` 核心庫與瀏覽器原生 WebCodecs 技術，繞過傳統伺服器轉碼的頻寬限制，直接調用本機硬體算力進行高效壓縮。
- 🔒 **本機沙盒極致隱私**：資料不落地保證。所有影片與照片轉碼皆在瀏覽器的沙盒環境中完成，絕不上傳至任何外部伺服器，完美捍衛商業機密與個人隱私。
- 👁️ **即時影音預覽劇院**：
  - 壓縮完成後（包含完美符合預期或容量溢出警告），自動解鎖內置的**純黑極簡影片播放器**。
  - 使用者可在下載前先觀看畫質細節、確認音軌，並掌握精確至小數點後兩位的真實產出大小（MB）。
- 📊 **智慧碼率與動態最佳化**：
  - 內建智慧容量調配算法，一鍵適配 JIRA（10MB）等常見系統限制。
  - 提供多級最高解析度限制（1080p 藍光、720p 行動標準、480p 標清速傳）。
  - 支援自動智慧計算與自訂鎖定指定碼率（Bitrate），可靈活移除音訊軌以極大化視訊流。
- 📸 **雙棲媒體支援**：完美適配影片（🎥）與照片（📸）雙重壓縮模組，提供一致流暢的極致體驗。

## 🔬 編碼支援矩陣 (Transcoding Matrix)

| 輸出編碼格式 (Codec) | 壓縮效率與體積 | 相容性評級 | 適用場景 |
| :--- | :--- | :--- | :--- |
| **H.264 / AVC** | 標準壓縮率 | 🌟🌟🌟🌟🌟 (完美相容) | 社群軟體傳輸、舊型裝置播放、萬用格式首選 |
| **H.265 / HEVC** | 極高（比 H.264 縮小約 50%） | 🌟🌟🌟🌟 (生態圈限制) | Apple 生態圈、大螢幕 1080p 深度節省空間 |
| **VP9** | 高 | 🌟🌟🌟🌟 (網頁友好的) | YouTube 核心編碼、Chrome & Android 行動端 |
| **AV1** | 次世代最高（體積最小） | 🌟🌟🌟 (效能吃重) | 追求極致微小體積，但需本機有較強硬體解碼力 |

## 🎨 設計語彙 (Apple Design System)

本專案深度貫徹 **Apple 核心設計系統樣式**，打造精緻如博物館展廳的行銷與工具界面：
- **視覺重心**：移除所有不必要的 UI 裝飾（無漸層、無大範圍投影、無卡片全域邊框），讓媒體成品（Artifact）本身成為視覺主角。
- **色彩規範**：全站以純白（`#ffffff`）、紙質偏置灰（`#f5f5f7`）與純黑（`#000000`）構築畫布，僅以單一**互動標誌藍 (Action Blue `#0066cc`)** 引導點擊。
- **微小互動**：所有按鈕皆綁定系統級微互動 `transform: scale(0.95)` 觸壓回饋。
- **精準排版**：採用 `SF Pro Display`（大標題）與 `SF Pro Text`（內文），在顯示尺寸上套用**負字距負值（Negative letter-spacing）**，重現極具張力的緊湊字體Cadence。

## 🛠️ 技術堆疊

- **前端框架**：React (Hooks 核心維護架構)
- **硬體轉碼核心**：`mediabunny` (整合 WebCodecs API 高階抽象管線)
- **樣式系統**：Scoped CSS & 磨砂玻璃濾鏡 (`backdrop-filter: blur`)
- **生命週期管理**：自動對 Blob 與 `URL.createObjectURL` 進行記憶體釋放（Revoke），防止前端瀏覽器記憶體溢漏（Memory Leak）。
