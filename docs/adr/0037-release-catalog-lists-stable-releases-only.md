# ADR-0037：版本與下載只列出正式 Release

## 狀態

已接受

## 背景

`main` 的 GitHub Pages 版本是可持續變動的預覽版，而版本目錄的目的是讓使用者選擇固定、可驗證並可回退的發布版本。將兩者混在同一列表容易誤導使用者安裝未固定的版本。

## 決策

「版本與下載」畫面只呈現正式 `vX.Y.Z` GitHub Release。`main` 預覽版只在目前應用程式的版本徽章標示「預覽版」，不在版本目錄提供 EXE、PWA 安裝或下載選項。

## 後果

- 版本目錄的每個項目都具有不可變 tag、Release notes、checksum 與可回退的資產。
- 預覽版仍可由開發者與測試者使用，但不會被一般使用者誤認為穩定發行物。
- `releases.json` 只收錄正式 release，並依版本或發布日期由新至舊排序。
