# ADR-0010：MOV 相容性先以 Chrome 驗收

## 狀態

已接受

## 背景

MOV 可否轉碼受瀏覽器解碼能力影響。現階段只需要為 Chrome 建立可執行且可重複的驗收範圍。

## 決策

以最新版 Chrome 驗收 macOS 產出的短片：

- H.264 MOV 必須可加入佇列、完成壓縮，且輸出 MP4 可播放。
- HEVC MOV 在 Chrome 可解碼時，必須同樣成功；無法解碼時，必須顯示明確的相容性錯誤，不得靜默失敗。

Safari 與其他瀏覽器不納入本次驗收範圍。

## 後果

- 本次實作與測試聚焦於 Chrome。
- README 的瀏覽器相容性敘述仍須如實保留「依瀏覽器與 codec 而異」的限制。
