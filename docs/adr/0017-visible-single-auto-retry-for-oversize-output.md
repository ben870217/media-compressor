# ADR-0017：超標輸出建立可見且只執行一次的自動重試任務

## 狀態

已接受

## 背景

原本超過目標容量時，系統在同一任務內隱性重跑多次。使用者無法辨識重試狀態，也無法保留各次輸出以比較。

## 決策

- 首次壓縮超過目標容量時，原任務標示為「完成，但超過目標」，保留結果且可下載。
- 系統在佇列中新增一個明確標示「自動重試 1/1」的任務，使用更低碼率。
- 重試任務在同一輪隊列中自動處理。
- 自動重試任務不得再建立任何後續重試任務。

## 後果

- 每個壓縮嘗試都有可見狀態與獨立結果。
- 使用者可比較原始壓縮結果與較低碼率的重試結果。
- 隊列排程需能在執行中接納新增任務，且不以固定快照決定全部工作。
